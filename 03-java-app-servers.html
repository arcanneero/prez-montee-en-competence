Java EE

Aide ou complexité ?

Java EE c'est quoi ?

Trois ensembles / sous ensemle de spécifications

  • Java Standrd Edition - Java SE
  • Java Enterprise Edition - Java EE
  • Java Mobile Edition - Java ME
  • Un ensemble de spécifications dans une version données
  • J2E, JEE sont d'anciennes appellations
  • Java SE - Implémentation de base
  • Java EE - Java SE plus des composant dit d'entreprise
  • Java ME - Adaptation légère pour la mobilité + éléments comme la gestion du temps real

Zoom sur

  • JPA - Persistence
  • JTA - Transaction
  • EJB - Transaction
  • JAX-RS - API Rest
  • JMS - Messaging
  • CDI - Injection de dépendance
  • etc.
